# DakarEaseU — Prompts Logo IA

**Concept retenu : "La Clé-Chapeau"**
La tête (bow) de la clé prend la forme d'un chapeau de diplôme.
Les dents de la clé sont des silhouettes d'immeubles miniatures (skyline de Dakar).
Les petites clés secondaires sont des accents décoratifs discrets.

**Couleurs de marque**
- Bleu marine profond : `#1E3A8A`
- Vert émeraude : `#10B981`
- Ambre / or : `#F59E0B`

**Outil recommandé (gratuit)** : [Ideogram.ai](https://ideogram.ai) — 10 générations/jour gratuites.
**Alternative gratuite** : Microsoft Designer / Bing Image Creator (illimité, backend DALL-E 3)

---

## Prompt 1 — Icône seule (app icon, sans texte)

```
A premium illustrative app icon for a student super-app. The central element is a large ornate key where the bow (circular top part of the key) is shaped like and morphs seamlessly into a graduation cap — the cap's flat board forms the top of the key bow, and the tassel hangs naturally from its corner. The key blade (shaft and teeth) points downward, and the teeth are stylized as tiny detailed building silhouettes — a miniature city skyline of Dakar, with varied heights and architectural details. The key is rendered in deep navy blue (#1E3A8A) with rich metallic engraving details and soft inner shadows. The graduation cap portion has subtle gold (#F59E0B) trim and tassel. The miniature buildings on the key teeth are in emerald green (#10B981) with fine window details. Two very small decorative skeleton keys float symmetrically in the background, slightly faded, acting as subtle accents. The overall composition is centered on a clean white background, slightly three-dimensional with layered depth, fine details, and a premium illustrative quality. No text. Square 1:1 format.
```

**Paramètres Ideogram** : Style → Illustration · Aspect ratio → 1:1 · Magic prompt → Off

White background. Horizontal layout 3:1. Premium illustrative brand logo quality, clean enough to scale down to 32px.

**Paramètres Ideogram** : Style → Illustration · Aspect ratio → 3:1 · Magic prompt → Off

## Conseils d'utilisation

1. **Ideogram.ai** → coller le prompt, désactiver "Magic Prompt", Style = Illustration.
2. **Microsoft Designer** → [designer.microsoft.com](https://designer.microsoft.com) → "Generate an image" → coller le prompt. Gratuit et illimité.
3. **Si les dents-immeubles ne sortent pas bien** : ajouter en fin de prompt → `the key teeth are clearly visible as tiny building silhouettes, architectural details, skyline shape`
4. **Fond transparent** : après génération → [remove.bg](https://remove.bg) pour isoler avant intégration dans l'app.
5. **Itérations** : générer 4–6 variantes, garder la meilleure composition, puis demander des variantes de couleurs.
